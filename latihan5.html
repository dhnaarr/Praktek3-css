Belajar CSS

Apa itu CSS?

CSS adalah singkatan dari "Cascading Style Sheets" atau Lembar Gaya Berjenjang dalam bahasa Indonesia.
Ini merupakan bahasa stylesheet yang digunakan dalam pengembangan web untuk menggambarkan presentasi dan format dari dokumen yang ditulis dalam HTML (Hypertext Markup Language) atau
XML (eXtensible Markup Language). CSS bertanggung jawab dalam mendefinisikan bagaimana konten sebuah halaman web harus ditampilkan pada perangkat dan ukuran layar yang berbeda.

Bagaimana Cara Menggunakan CSS?

Ada tiga cara menggunakan CSS

  1. Inline Styles (Gaya Inline)
    Gaya inline mengacu pada penggunaan properti CSS langsung di dalam tag HTML. Ini berarti Anda menambahkan gaya secara langsung pada elemen HTML yang bersangkutan dengan menggunakan atribut "style".

  2. Internal Styles (Gaya Internal)
    Gaya internal diimplementasikan dengan menempatkan kode CSS di dalam elemen style di dalam tag head dari dokumen HTML. Ini memungkinkan Anda mendefinisikan gaya untuk beberapa elemen di satu halaman.
